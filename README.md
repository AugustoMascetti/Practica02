Practica02 - Ciencia de datos en organizaciones

Descripción: en este repositorio esta la resolución del ejercicio 10 del trabajo práctico.
Grafica las tablas de posiciones por ronda y la tabla final a partir de los datos del PDF de la practica.

Requisitos:

-Tener instalado Python 3.

-Instalar JupyterLab ejecutando el siguiente comando en una terminal(ver en la ejecucion): pip install jupyter[lab]

Ejecución:

1. Clonar el repositorio con el siguiente comando en una terminal: git clone https://github.com/AugustoMascetti/Practica02.git

2. Acceder a la carpeta del proyecto: cd Practica02

3. Instalar Jupyter lab y después ejecutarlo usando el siguiente comando: jupyter lab

4. Una vez que se abre el navegador, hay que ir a la carpeta llamada 'notebooks/' y entrar al archivo 'Consigna10Simulacion.ipynb'.

5. Ejecutar todas las celdas para ver los graficos.

Estructura

- El código .py se encuentra en la carpeta 'src/', en donde esta la función que permite la simulación de la competencia de comida.

- El notebook usa la función de 'src/'-
